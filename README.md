## Installation

Install Python 

```bash
pkg install python 
```
Install Git

```bash
pkg install git
```
Clone Repostory

```bash
git clone https://github.com/isk-d3v/lsk
```

Go to repostory

```bash
cd lsk
```

Start the python script

```bash
python lsk.py
```

Ur done !
